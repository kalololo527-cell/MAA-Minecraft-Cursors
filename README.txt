About this project:

This cursor set was designed as a small Arabic creative project inspired by the world of Minecraft.
I wanted to turn some familiar items from the game, such as the Name Tag, TNT, and the Trident, into simple mouse cursors that people can use on their desktop.

Installation Instructions (Manual):

Extract the file if it is compressed.

Open the main folder of the package.

You will find three folders:

Select

Resize

Other

Open your mouse settings in Windows.

Go to the Pointers tab.

Choose the pointer type you want to change (e.g., Normal Select or Link Select).

Click Browse.

Go to one of the folders in the package and select the appropriate file.

How to select the appropriate file: You will find them with the same name inside the Pointers tab.

Click Apply, then OK to apply the change.
